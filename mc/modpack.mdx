We run a custom modpack built around Create and Cobblemon with many quality of life mods and extras for added immersion.

This pack also comes with a completely custom progression system made exclusively for Trixtail Tavern!

## Installation

Go to [https://modrinth.com/modpack/the-cyanist-empire](https://modrinth.com/modpack/the-cyanist-empire) and click Download.

### Quick Install

For the smoothest experience, installing with Modrinth App is recommended for one-click installs and automatic updates!

If you already have Modrinth App installed, you can simply search for The Cyanist Empire under modpacks.

### Manual Install

Want to use your own launcher? Look up "mrpack to zip" and use one of the available online tools.

This will convert the downloaded modpack to a zip file containing the bundled mods and configuration files.

Install Neoforge for 1.21.1 then extract the zip into your Minecraft installation folder and you're good to go!
