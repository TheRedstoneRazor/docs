Numerous minor tweaks have been made to various aspects of Minecraft to rebalance the experience in line with our vision.

# Minor Changes

**Stackable Utilities:** Potions and typically non-stackable food items like mushroom stew now stack to 16.

**Villager Trade Rebalance:** The villager trading rebalance experiment is enabled by default.

# Block Drops

The yield from certain blocks has been adjusted, mainly with regards to the Fortune enchantment to rebalance resource scarcity.

### Copper Ore

Copper has been adjusted to drop once per ore, in line with other metal ores. The cut copper recipe has been adjusted to balance this change while allowing tools and Create components to be more demanding to craft.

### Redstone Ore & Lapis Lazuli Ore

For other ores that can drop multiple items without enchantments, the maximum number of items dropped has been slightly reduced.

#### Lapis Lazuli Ore

To be consistent with redstone, Fortune now increases the maximum number of items dropped by 1 per level, instead of 1 per item dropped per level.

| Ore | Old Max | New Max |
| --- | --- | --- |
| Redstone | 4-5 | 2-3 |
| Redstone (Fortune III) | 4-8 | 2-6 |
| Lapis Lazuli | 4-9 | 3-6 |
| Lapis Lazuli (Fortune III) | 4-36 | 3-9 |

### Amethyst Cluster

Fully grown amethyst clusters now drop one amethyst shard to encourage setting up amethyst farms around geodes.

### Farmland Crops

Harvesting farmland crops (Wheat, potatoes, etc.) using a hoe enchanted with Fortune now increases the yield.
